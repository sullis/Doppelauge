  42. def change(converter: PartialFunction[(String, JValue), (String, Any)]): JValue

    Less verbose way to map content of a json object

    Less verbose way to map content of a json object

    Definition Classes
    JValue
    Example: 

    1.  // Example 1:

 json.change {
   case ("adouble", value: JValue) => "adouble" -> 9.87
 }

 // Is the same as:

 J(
   json.asMap.map {
     case ("adouble",   value: JValue) => "adouble" -> 9.87
     case (key: String, value: JValue) => key       -> value
   }
 )


 // Example 2:

 json.change {
   case ("anint", _)               => J.___removeThisField
   case ("adouble", value: JValue) => "adouble" -> 9.87
 }

// is the same as:

 J(
   json.asMap.map {
     case ("anint", _)                 => J.___removeThisField
     case ("adouble",   value: JValue) => "adouble" -> 9.87
     case (key: String, value: JValue) => key       -> value
   }.filter(_ != J.___removeThisField)
 )
    Note

    There is no warning or exception thrown if trying to change a non-existing field.

    See also

    JsonChanger for a much safer way to change json values.
